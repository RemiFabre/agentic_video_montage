# agentic_video_montage

A reference recipe for co-creating short-form social-media video montages with an AI coding agent (Claude Code, Codex, Cursor, etc.).

The interesting work in this repo lives in [`AGENTS.md`](AGENTS.md). It's the system-prompt-style brief I'd hand to an agent before starting a new montage session — pipeline, conventions, ffmpeg/whisper recipes, things to *not* do, and the iteration discipline that keeps the back-and-forth tight.

If you're a human, read `AGENTS.md` to understand how the loop works. If you're an agent, read it twice.
